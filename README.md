# 🎤 판다마켓

> 일상의 모든 물건을 거래하는 마켓 플랫폼

## 🌟 프로젝트 소개

판다마켓(Panda Market)은 누구나 손쉽게 물건을 구매하고 판매할 수 있는 온라인 마켓 플랫폼입니다.
회원가입과 로그인 기능을 통해 개인 맞춤형 서비스를 제공하며, 인기 있는 베스트 상품을 한눈에 볼 수 있습니다.
또한 이용자 간 자유롭게 소통할 수 있는 자유게시판을 운영하여 활발한 커뮤니티 환경을 제공합니다.
